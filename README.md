# Personal Portfolio Website

A simple and responsive personal portfolio website built using **HTML** and **CSS**. This project showcases a clean and minimal design with easy navigation between the main pages.

## Features

* Responsive and modern layout
* Home page introducing the portfolio
* Contact page for visitor communication
* Clean and beginner-friendly HTML & CSS structure

## Project Structure

```
portfolio/
│── home.html
│── contact.html
│── style.css
```

## Technologies Used

* HTML5
* CSS3

## Pages

* **Home** – Displays the main portfolio content and introduction.
* **Contact** – Provides contact information or a contact form for visitors.

## Purpose

This project was created to practice front-end web development using HTML and CSS while building a simple personal portfolio website.

## How to Run

1. Download or clone this repository.
2. Open `home.html` in your web browser.
3. Navigate to the Contact page using the website menu.
